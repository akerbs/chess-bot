# Chess Bot Web App

Веб-приложение для игры в шахматы через Telegram Web App.

## Запуск

### Вариант 1: Простой HTTP сервер

```bash
cd webapp
node server.js
```

### Вариант 2: Использование nginx или другого веб-сервера

Настройте веб-сервер для раздачи статических файлов из директории `webapp`.

### Вариант 3: Хостинг на GitHub Pages, Netlify, Vercel и т.д.

Загрузите файлы из директории `webapp` на любой статический хостинг.

## Настройка

В файле `.env` добавьте:

```
WEB_APP_URL=https://your-domain.com
```

Или для локальной разработки:

```
WEB_APP_URL=https://your-ngrok-url.ngrok.io
```

## Особенности

- Полноценная шахматная доска с drag and drop
- Подсветка возможных ходов
- Анимации перемещения фигур
- Адаптивный дизайн
- Поддержка темной темы Telegram
